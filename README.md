# 設備点検管理システム

## 概要
施設管理の現場を想定した設備点検管理システムです。
設備情報、点検履歴、点検期限、状態管理を行い、対応が必要な設備を把握しやすくすることを目的としています。

## 作成目的
株式会社FMシステム様の事業内容を踏まえ、施設管理・保全管理に近いテーマとして作成しました。
求人票で求められていた C#、HTML/CSS、JavaScript、SQL Server を使用し、業務システム開発を想定しています。

## 使用技術
- C#
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server LocalDB
- HTML / CSS
- JavaScript
- Bootstrap

## 主な機能
- 設備情報の登録・編集・削除・一覧表示
- 設備名、状態、点検期限による検索・絞り込み
- 設備状態の色分け表示
- 次回点検日の期限超過・7日以内の点検予定の強調表示
- 設備ごとの点検履歴登録
- 点検結果に応じた設備状態の自動更新
- 状態別件数や期限超過件数を確認できるダッシュボード
- 削除前の確認メッセージ表示

## 工夫した点
- 設備状態を色付きラベルで表示し、対応が必要な設備を見つけやすくしました。
- Controller側で検索条件を受け取り、LINQを使ってSQL Serverから条件に合う設備を取得するようにしました。
- 設備と点検履歴を1対多で紐づけ、設備ごとに過去の点検結果を確認できるようにしました。
- 点検履歴登録時に、点検結果に応じて設備状態や次回点検日を更新するようにしました。
- ダッシュボードで状態別件数や期限超過件数を表示し、優先的に確認すべき設備を把握できるようにしました。
